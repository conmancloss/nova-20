How It Works
Redis-backed
All blocks are stored in Upstash Redis under the key nova:blocked and fetched by every Nova client on load and every 2 minutes.
Proxy intercept
Before the UV proxy navigates to a URL, nova.js checks it against the cached block list. Blocked URLs show an access-denied screen instead of loading.
Auto-logged
Every block/unblock action appears in the Activity log and in the Broadcast & Block History on the Broadcast panel.
